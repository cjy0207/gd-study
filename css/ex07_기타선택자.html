1. 속성 선택자

선택하고자하는 요소내에 이미 생성된 속성을 활용하여 선택 선택자 뒤에 []를 이용 

        선택자[속성=속성값] {...} : 속성=속성인 경우
        선택자[속성~=속성값] {...} : 속성에 속성값이 포함된 태그
        선택자[속성|=속성값] {...} : 
        선택자[속성^=속성값] {...}

2. 자손/후손 선택자

많은 요소들이 중첩되어 있는 경우 자손 : 바로 하위의 요소들 후손 : 하위요소들 전부 

        자손 선택자 : >
        a > b{
            스타일 속성 : 값;
        }
        후손 선택자 : 공백
        a b {
            스타일 속성 : 값;
