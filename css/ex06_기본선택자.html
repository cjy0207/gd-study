선택자란?

특정 html요소를 선택할때 사용하는 기능

1. 모든 선택자 : *

2. tag 선택자 : 태그명

3. id 선택자 : #아이디명

현재 문서의 특정 html요소 중 한개만 선택하고자 할 때 사용. 단, id속성은 고유한 아이디 값을 부여해야함

  1. 아이디 선택자 테스트
  2. 아이디 선택자
  3. 아이디 선택자
  4. 아이디 선택자

4. 클래스 선택자 : .클래스명

해당 문서내에 원하는 요소 여러개를 함께 선택하고자 할 떄 사용

  1. 아이디 선택자 테스트
  2. 아이디 선택자
  3. 아이디 선택자
  4. 아이디 선택자
