자바스크립트 개요

HTML 페이지의 동작을 사용자의 요구에 맞도록 수행 가능
브라우저 언어
인터프리터 방식의 언어

자바스크립트의 특징

  • 접근제한자 개념없음
  • 변수 선언이 필요없음. 초기화시 자료형 결정
  • 웹 환경에 특화된 언어 -> 보안등의 이유로 기능 제한됨
  • 보안에 취약 : 소스가 외부에 공개

인라인방식

<태그 on이벤트명 = "자바스크립트소스()">
<a href="javascript:소스">실행</a>

링크태그 출력 링크태그 console 출력
