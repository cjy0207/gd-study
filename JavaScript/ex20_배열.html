배열

자바스크립트에서는 어떤 자료형이든 하나의 배열에 저장 가능
크기에 제한 없음. 자바의 ArrayList와 비슷. 리스트에 가까움

배열에 값 저장하기

배열 주요 메소드

배열.indexOf(요소) : 배열에서 요소의 위치값 리턴


배열.concat(배열,배열...) : 배열에 배열 추가하기


배열.reverse() : 배열의 요소를 역순으로 변경


배열.sort() : 배열의 요소를 순서대로 정렬


배열.pop(), push() : 배열의 요소를 조회, 추가


배열.unshift(추가요소): 배열의 맨앞에 요소 추가, 배열의 크기반환
배열.shift() : 배열의 맨 앞의 요소 제거, 제거된 요소 리턴


배열.slice(시작인덱스, 끝인덱스+1) : 시작인덱스부터 끝인덱스까지 리턴 배열.splice(시작인덱스, 제거갯수, [추가할 값]) : 배열의 요소 제거 및 추가 ->치환
